Spike: Brain Simulator Parallelization
Spike aims to bring performance enhancements to the nengo project through parallelization. We aim to run ensembles concurrently and split up ensembles to evenly distribute CPU load. We further plan to distribute the ensembles across machines using a fast transport protocol 0MQ.
Lifelinevis
Lifelinevis is a set of dynamic visualizations that demonstrate message propagation within QNX. The project utilises mTags, lightweight meta-data that augments QNX inter-process communication through biomarker-line behavior. You can read more about mTags here: research paper. You can also read more about Lifelinevis here: report.
